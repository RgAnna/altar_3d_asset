## Crystal Altar (Dungeon Environment Asset)
- **Polygons:** 35,600 tris
- **Texture Resolution:** 2048x2048
- **Environment:** Underground / Dungeon
- **Purpose:** 
  - Ritual zone
  - Quest interaction object
  - Boss arena centerpiece
  - Magical power source

## Asset Description
The Crystal Altar is a high-quality environment asset designed for dungeon and underground locations.  
It features a massive stone altar structure infused with a glowing crystal core, suitable for fantasy, sci-fi, or hybrid RPG projects.

The asset is optimized for real-time use while preserving visual fidelity and atmosphere.

# Fantasy / Sci-Fi Ассет: Алтарь с Кристаллом (Подземелье)

## Характеристики ассета

### Алтарь с Кристаллом
- **Полигоны:** 35 600 треугольников
- **Разрешение текстур:** 2048x2048
- **Окружение:** Подземелье
- **Назначение:**
  - Ритуальная зона
  - Квестовый объект
  - Центральный элемент арены босса
  - Источник магической энергии

## Описание
Алтарь с кристаллом — это детализированный ассет окружения для подземелий и мрачных локаций.  
Массивная каменная конструкция с энергетическим кристаллом в центре подходит для fantasy, sci-fi и hybrid RPG проектов.

Ассет оптимизирован для использования в реальном времени с сохранением визуального качества.
